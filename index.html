Sphinx Doc – MATLAB Example 1¶

Examples of auto-documenting MATLAB source files with Sphinx, which lets you create beautiful documentation. The source for this project is found at https://github.com/joeced/sphinx-docs-matlab-example-1.

The file structure for this project is, where source files are located in the src folder.:

docs
    conf.py
    index.rst
src
    @ClassFolder
        a_static_func.m
        ClassFolder.m
        classMethod.m
    +package
       func.m
    submodule
       ClassInheritDifferentModules.m
       TestFibonacci.m
    ClassBySource.m
    f_example.m

In conf.py, we have setup the use of sphinxcontrib-matlabdomain with:

matlab_src_dir = os.path.abspath('..')
primary_domain = 'mat'
extensions = ['sphinx.ext.autodoc', 'sphinxcontrib.matlab']
